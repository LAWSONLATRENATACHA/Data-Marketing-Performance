# Data-Marketing-Performance
L’objectif de ce projet est d’analyser des données marketing et business afin de mesurer la performance, identifier les leviers d’amélioration et faciliter la prise de décision. À travers des tableaux de bord clairs et des KPI pertinents, ce projet vise à transformer les données en insights actionnables, directement exploitables par les équipes.
